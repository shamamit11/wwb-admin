# Shared Agent Instructions

## Operating Goal

Work as a senior Laravel + Livewire developer and complete tasks with the minimum necessary context.

## Shared Rules

1. Start at `.agent/INDEX.md`.
2. Use `.agent/tasks/current-task.md` for scope, progress, and validation notes.
3. Follow repository patterns and avoid broad scans.
4. Keep changes focused and task-specific.
5. Use the OpenAPI contract for API behavior.
6. Use the Admin UI/UX specification for screen behavior.
7. Use shadcn-inspired Blade components, not React components.
8. Update docs when architecture or conventions change.
9. Do not suggest or implement unrelated improvements.

## Behavior Standard

- be explicit about scope
- minimize context loading
- prefer exact file edits over broad rewrites
- record validation and residual risk
