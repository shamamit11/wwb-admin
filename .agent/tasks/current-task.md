# Task: Improve Draft Review Admin Page UI/UX

Status: Completed

## Goal

Improve the Draft Review index so it reads as an editorial review queue, with clearer primary review actions, stronger title/excerpt hierarchy, and easier-to-scan provenance metadata for AI-generated drafts.

## Background

The current Draft Review screen is functional and already scoped to AI-generated draft posts, but its table still behaves like a generic post list. Editors should be able to immediately identify the review action, assess draft provenance, and decide which post to open next without drilling into a row action menu first.
